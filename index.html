Pasos para utilizar GIT

  1. Crear un repositorio en GitHub.
  2. Clonar el repositorio en tu máquina local.
  3. Crear una rama para cada colaborador.
  4. Realizar cambios en el código y hacer commits.
  5. Fusionar las ramas para integrar los cambios.
